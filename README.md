# Git Comandos
- git clone (clona um repositório github para pc)
- git status (mostra o status do repositório local)
- git add nome-arquivo (adicionar no stage changes o arquivo)
- git add . (adicionar todos os arquivos do path no stage changes)
- git checkout -b (cria uma nova branch)
- git commit -m (salva os arquivos no repositório local)
- git push (enviar os dados repositório local para o respositório remoto)
